# 🌡️ Sistema de Monitoramento de Ambiente e Estoque (C)

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conhecimentos em **estruturas de decisão (if/else)** na linguagem C.

## 📌 Sobre o projeto

O programa simula um sistema simples de monitoramento que analisa três variáveis importantes:

* Temperatura
* Umidade
* Nível de estoque

Com base nos valores inseridos pelo usuário, o sistema avalia cada condição e retorna mensagens informando se os parâmetros estão dentro do esperado ou se exigem atenção.

## 🧠 Conceitos aplicados

* Estruturas de decisão (`if` e `else`)
* Entrada de dados com `scanf`
* Saída de dados com `printf`
* Tipos de dados (`float` e `unsigned int`)
* Comparações lógicas

## ⚙️ Funcionamento

O programa solicita ao usuário:

1. A temperatura atual
2. O nível de umidade
3. A quantidade de estoque

E realiza as seguintes verificações:

* 🌡️ **Temperatura**

  * Acima de 30 → Temperatura alta
  * Caso contrário → Dentro dos parâmetros

* 💧 **Umidade**

  * Acima de 50 → Umidade elevada
  * Caso contrário → Dentro dos parâmetros

* 📦 **Estoque**

  * Acima de 1000 → Estoque normal
  * Abaixo ou igual → Estoque abaixo do mínimo



(Este projeto faz parte da minha jornada de aprendizado em programação e desenvolvimento de sistemas, com foco em aplicar conceitos teóricos na prática.)

# 🌡️ Environment and Inventory Monitoring System (C)

This project was created to practice and reinforce my understanding of **decision structures (if/else)** in the C programming language.

## 📌 About the project

The program simulates a simple monitoring system that analyzes three key variables:

* Temperature
* Humidity
* Stock level

Based on the user’s input, the system evaluates each condition and provides feedback indicating whether the values are within acceptable limits or require attention.

## 🧠 Concepts applied

* Decision structures (`if` and `else`)
* Input handling with `scanf`
* Output using `printf`
* Data types (`float` and `unsigned int`)
* Logical comparisons

## ⚙️ How it works

The program asks the user for:

1. Current temperature
2. Humidity level
3. Stock quantity

It then performs the following checks:

* 🌡️ **Temperature**

  * Above 30 → High temperature
  * Otherwise → Within acceptable range

* 💧 **Humidity**

  * Above 50 → High humidity
  * Otherwise → Within acceptable range

* 📦 **Stock**

  * Above 1000 → Stock level is normal
  * Otherwise → Below minimum level


(This project is part of my learning journey in programming and systems development, focused on turning theoretical knowledge into practical experience.)
